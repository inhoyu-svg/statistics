테이블 시각화 케이스

기본 테이블, 줄기-잎 그림, 카테고리 행렬, Corruption 효과 등 다양한 테이블 케이스

▶ 기본 테이블 (basic-table)

T1 - 기본 테이블

가장 단순한 basic-table 형태

T2 - 도수분포표

showTotal: true, showMergedHeader: false

T2a - 탈리마크 도수분포표

탈리마크(///)와 도수가 함께 표시된 표

T3 - 이원분류표

병합 헤더가 있는 이원분류표

T4 - 변량 테이블

showGrid: false로 격자선 없이 데이터만 표시

T4a - 변량 테이블 (문자 변수)

a, b 등 문자 변수가 포함된 변량 테이블

▶ 줄기-잎 그림 (stem-leaf)

T5 - 줄기-잎 그림 (단일)

단일 데이터셋의 줄기-잎 그림

T6 - 줄기-잎 그림 (비교)

두 그룹 비교 줄기-잎 그림

▶ 카테고리 행렬 (category-matrix)

T7 - 카테고리 행렬

헤더가 왼쪽으로 이동한 교차표 형태

T7a - 카테고리 행렬 (단일 행)

한 행만 있는 간단한 카테고리 행렬

T7b - 카테고리 행렬 (다중 행)

여러 행이 있는 복잡한 카테고리 행렬

▶ Corruption 효과 (찢김)

T8 - 찢어진 테이블

Corruption 효과가 적용된 테이블

T9 - 찢어진 줄기-잎

Corruption 효과 + 줄기-잎 그림

T10 - 찢어진 이원분류표

Corruption 효과 + 이원분류표

▶ 셀 애니메이션 (cellAnimations)

T11 - 단일 셀 애니메이션

특정 셀 하나에 깜빡임 효과

T12 - 행 범위 애니메이션

특정 행 전체에 애니메이션

T13 - 열 범위 애니메이션

특정 열 전체에 애니메이션

T14 - 영역 범위 애니메이션

여러 셀 영역에 애니메이션 (rowStart/End, colStart/End)

T15 - 2개 그룹 애니메이션

서로 다른 두 영역이 동시에 애니메이션

T16 - 3개 그룹 애니메이션

세 개의 서로 다른 영역이 동시에 애니메이션

T17 - 이원분류표 셀 애니메이션

이원분류표에서 특정 셀 하이라이트

T18 - 카테고리 행렬 셀 애니메이션

카테고리 행렬에서 특정 영역 하이라이트

T19 - 줄기-잎 그림 셀 애니메이션

줄기-잎 그림에서 특정 잎 하이라이트
