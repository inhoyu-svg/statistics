# 📊 통계 시각화 도구

통계 데이터를 입력하면 자동으로 **도수분포표, 히스토그램, 상대도수 다각형**은 물론 **이원분류표, 줄기-잎 그림, 카테고리 행렬** 등 다양한 통계표를 생성해주는 웹 애플리케이션입니다.

## 🚀 바로 사용하기

**웹사이트**: [https://math-statistics-library.vercel.app](https://math-statistics-library.vercel.app)

설치 없이 브라우저에서 바로 사용할 수 있습니다!

## ✨ 주요 기능

### 📋 다양한 테이블 타입 지원
- **도수분포표**: 데이터만 입력하면 계급, 도수, 상대도수, 누적도수 자동 계산
- **이원 분류표**: 행(카테고리)과 열(그룹)로 데이터 분류 (예: 혈액형별 남녀 분포)
- **카테고리 행렬**: 다중 카테고리 데이터 표현
- **줄기-잎 그림**: 데이터의 분포를 줄기와 잎으로 시각화

### 📊 차트 시각화
- **히스토그램**: 데이터의 분포를 막대 그래프로 표시
- **상대도수 다각형**: 데이터의 경향성을 선 그래프로 표시
- **다각형 색상 선택**: 4가지 색상 프리셋 제공
- **막대 위 값 표시**: 히스토그램 막대 위에 값(상대도수 또는 도수) 표시
- **말풍선 표시**: 특정 막대에 커스텀 텍스트 말풍선 표시
- **수직 파선 표시**: 특정 위치에 수직 파선 추가
- **합동 삼각형 표시**: 경계값 기준 삼각형 시각화

### 📈 데이터 분석
- **실시간 통계**: 평균, 중앙값, 범위 등 기본 통계량 자동 계산
- **다중 데이터셋**: 여러 데이터셋을 동시에 관리하고 비교
- **테이블 셀 편집**: 모달에서 셀 값을 변수로 치환 (빈칸은 `_` 입력)

### ⚡ 고급 기능
- **빈 구간 압축**: 데이터가 없는 계급 구간을 이중물결표(≈)로 자동 압축
- **Y축 커스텀 간격**: 상대도수/도수 모드 전환 및 커스텀 간격 설정
- **격자선 토글**: 가로/세로 격자선 개별 표시/숨김
- **축 라벨 커스터마이징**: X축/Y축 라벨 및 끝 라벨 자동 표시

### 🎬 애니메이션 & 레이어 관리
- **애니메이션 제어**: 재생/일시정지/정지, 속도 조절 (0.5x~3.0x)
- **레이어 관리**: 차트/테이블 요소별 표시/숨김 및 순서 변경

### ⚙️ 테이블 커스터마이징
- **컬럼 설정**: 표시/숨김, 순서 변경, 정렬, 라벨 변경
- **상첨자 표시**: 첫 계급에 이상/미만 상첨자 표시 옵션
- **합계 행 토글**: 합계 행 표시/숨김

### 💾 데이터 저장
- **JSON 내보내기/불러오기**: 차트 데이터를 JSON으로 저장하고 불러오기

## 📖 사용 방법

### 1. 테이블 타입 선택

드롭다운에서 원하는 테이블 타입을 선택하세요:
- **도수분포표**: 숫자 데이터의 분포 분석 (기본값)
- **이원 분류표**: 두 변수의 교차 분류
- **카테고리 행렬**: 다중 카테고리 데이터
- **줄기-잎 그림**: 데이터 분포의 시각적 표현

### 2. 데이터 입력

데이터를 **쉼표(,)** 또는 **공백**으로 구분하여 입력하세요.

**입력 예시:**
```
2, 5, 8, 5, 2, 3, 7, 9, 4, 6, 5, 8, 3, 2, 7, 10
```

또는

```
2 5 8 5 2 3 7 9 4 6 5 8 3 2 7 10
```

### 3. 계급 설정 (도수분포표)

**계급 개수**: 데이터를 몇 개의 구간으로 나눌지 설정 (3~20 사이)
- 기본값: 5개
- 데이터가 많을수록 계급 개수를 늘리는 것이 좋습니다

**계급 간격** (선택사항): 각 구간의 크기를 직접 지정
- 비워두면 자동으로 계산됩니다
- 예: 간격을 2로 설정하면 0~2, 2~4, 4~6 형태로 생성

### 4. 도수분포표 생성

"**도수분포표 생성**" 버튼을 클릭하거나 **Ctrl + Enter**를 누르세요.

### 5. 결과 확인

생성된 결과는 3가지로 표시됩니다:

#### 📊 통계 요약 카드
- 데이터 개수
- 최솟값 / 최댓값
- 범위
- 평균
- 중앙값

**Tip**: 각 카드에 마우스를 올리면 상세 설명을 볼 수 있습니다!

#### 📋 도수분포표
| 계급 | 계급값 | 도수 | 상대도수(%) | 누적도수 | 누적상대도수(%) |
|------|--------|------|-------------|----------|-----------------|
| 2~4  | 3.00   | 4    | 26.67%      | 4        | 26.67%          |
| 4~6  | 5.00   | 4    | 26.67%      | 8        | 53.33%          |
| ...  | ...    | ...  | ...         | ...      | ...             |

#### 📈 히스토그램 & 상대도수 다각형
- **파란색 막대**: 각 계급의 상대도수를 나타내는 히스토그램
- **주황색 선**: 데이터의 분포 경향을 나타내는 상대도수 다각형
- **초록색 테두리**: 데이터가 있는 구간 강조
- **축 끝 라벨**: X축 끝에 (계급), Y축 끝에 (상대도수) 또는 (도수) 자동 표시

## 💡 사용 팁

### 데이터 입력 시
- ✅ 숫자만 입력하세요 (음수, 소수점 가능)
- ✅ 최소 2개 이상의 데이터 필요
- ✅ 최대 10,000개까지 입력 가능
- ❌ 문자나 특수문자는 자동으로 제거됩니다

### 계급 설정 시
- 데이터 개수가 적으면 (10개 이하): 계급 개수 3~5개 권장
- 데이터 개수가 많으면 (100개 이상): 계급 개수 7~15개 권장
- 계급 간격은 데이터 범위에 맞게 자동 계산됩니다

### 결과 해석
- **평균 vs 중앙값**: 평균이 중앙값보다 크면 오른쪽으로 치우친 분포
- **히스토그램 모양**:
  - 종 모양: 정규분포에 가까움
  - 왼쪽 치우침: 높은 값이 많음
  - 오른쪽 치우침: 낮은 값이 많음

## 🔍 FAQ (자주 묻는 질문)

**Q: 데이터를 입력했는데 계급이 0부터 시작해요.**

A: 이 프로그램은 모든 계급을 0부터 시작하도록 설계되었습니다. 데이터가 없는 구간(도수 0)은 표에서 자동으로 숨겨지지만, 히스토그램에는 빈 공간으로 표시됩니다.

**Q: 계급 간격을 지정하면 어떻게 되나요?**

A: 계급 간격을 직접 입력하면 지정한 크기로 고정되어 계급이 생성됩니다. 비워두면 데이터 범위와 계급 개수에 맞춰 자동으로 계산됩니다.

**Q: 히스토그램에서 이중물결(≈) 표시는 무엇인가요?**

A: 데이터가 없는 빈 계급 구간이 3개 이상 연속되면 자동으로 압축되어 이중물결 패턴(≈)으로 표시됩니다. 이를 통해 차트가 더 깔끔하게 보이며, 실제 데이터가 있는 구간에 집중할 수 있습니다.

**Q: 히스토그램과 상대도수 다각형의 차이는 무엇인가요?**

A: 히스토그램은 각 구간의 데이터 개수(도수)를 막대로 표시하고, 상대도수 다각형은 각 구간의 중앙값을 연결한 선 그래프로 데이터 분포의 경향성을 보기 쉽게 표현합니다.
